blackjack

This project provides: * Flexible blackjack library blackjack-lib allowing to create a different blackjack games; * CLI tool to play a blackjack blackjack-cli locally.

Install from source

Latest sources 
python setup.py pytest && python setup.py install
blackjack-cli
